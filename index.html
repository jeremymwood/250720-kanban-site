Full-Stack Portfolio Project

Role-aware Kanban workflows with production-minded engineering.

Ishi demonstrates end-to-end product ownership: React + TypeScript frontend, Express + Prisma backend, PostgreSQL data model, role-based permissions, email verification, and hardened auth/session controls.

Key Highlights

  • Admin / Developer / Client role model with policy-enforced behavior.
  • Kanban board with status lanes, drag-and-drop, and modal-first issue workflows.
  • JWT auth with refresh token rotation, activation gates, and email verification.
  • Security hardening: CORS allowlist, Helmet, rate limits, input validation.
  • Operational reliability: health probes, request IDs, graceful shutdown handling.

Stack

  • Frontend: React, TypeScript, Vite
  • Backend: Node.js, Express, TypeScript
  • Database: PostgreSQL with Prisma ORM
  • Integrations: Twilio Verify, Cloudflare, Railway
